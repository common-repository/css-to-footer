== Description ==

CSS to Footer is a easy-weight plugin. It takes CSS files you chosen, and brings them to the footer to accelerate site loading speed. If your scores in Google pagespeed is't perfect try to use this plugin.

== Installation ==

1. After plugin activation, open your website page (Not in admin-mode!).
2. Press F12 and open <head> element.
3. You can see strings looking like this:
"link rel="stylesheet" id="wp-block-library-css" href="http://localhost/infinity/wp-includes/css/dist/block-library/style.min.css?ver=5.2.5" type="text/css" media="all""
4. You need to copy id ("wp-block-library-css" in this example).
5. And paste it in plugin's settings page(!without "",'' and spaces!).
